    ## Quicklook

    - トークンは「Primitive（Size コレクション）→ Semantic（Pattern コレクション）」の2層で管理しています
    - Pattern コレクションは **Standard** と **Comfortable** の2つのモードを持ち、UIの密度を切り替えられます
    - コンポーネントのサイズ・パディング・ギャップ・アイコンサイズはすべて Semantic トークンで制御します

    ## トークンの階層

    Garage のサイズトークンは2層構造で管理されています。Primitive 層で基礎となる数値スケールを定義し、Semantic 層でコンポーネントやレイアウトの文脈に応じた意味を与えます。

    {/* figma-frame: PSqBlA9pU7SHYbplxhS8q6/3058-586 | Size 概要 */}

    | 層 | コレクション | 役割 |
    | --- | --- | --- |
    | **Primitive** | Size | 2px〜280px の固定値スケール。すべてのサイズの基盤です。 |
    | **Semantic** | Pattern | コンポーネントの高さ・パディング・ギャップなど、UIパターンに応じた意味を持つ層です。Standard / Comfortable の2モードで密度を切り替えます。 |

    ## UI Pattern（モード）

    Pattern コレクションは以下の2モードを持ちます。

    | モード | 用途 |
    | --- | --- |
    | **Standard** | デスクトップ向けの標準密度。多くのプロダクトUIで使いましょう。 |
    | **Comfortable** | タッチ操作が主体のUI。モバイルやタブレットに適しています。 |
